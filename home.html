Welcome to BYR-Archive

A high-performance NPM package CDN proxy built in C.

Usage Examples

Replace http://localhost:8080 with your server's address.

1. Get Package Entry File (Latest Version)

To get the main entry file of the latest version of a package:

curl http://localhost:8080/react

2. Get Package Entry File (Specific Version)

To get the main entry file of a specific version:

curl http://localhost:8080/vue@3.3.4

3. Get Package Entry File (Version Range)

The server supports ^ and ~ version ranges. The following command will fetch the latest version in the 3.x series.

curl http://localhost:8080/vue@^3.0.0

4. Browse Package Contents

To view the file listing of a package directory, add a trailing slash:

# In your browser, navigate to:
http://localhost:8080/vue@3.3.4/

5. Get a Specific File

To get a specific file from within a package:

curl http://localhost:8080/react@18.2.0/umd/react.development.js
